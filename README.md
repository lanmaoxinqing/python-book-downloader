## description
小说下载器，目前支持 八八,笔趣,顶点

## usage
```shell
bd.py [-h] [-n NAME] [-c CHANNEL] [-opt OPTION] name

positional arguments:
  name         书名或URL

optional arguments:
  -h, --help   show this help message and exit
  -n NAME      书名或URL
  -c CHANNEL   0八八, 1笔趣, 2顶点
  -opt OPTION  0书名, 1URL
```

v3.6.2
